- Developed and deployed a news web application for quick personal updates using Django and PostgreSQL, deployed on Vercel.
- Database Management: Utilized Railway to create and manage the PostgreSQL database, benefiting from its ease of deployment and robust parallel processing capabilities.

- Frontend Integration: Enhanced user interaction by integrating Bootstrap components into the frontend.
